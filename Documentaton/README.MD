# What is OpenPilot

Well, OpenPilot is an Operating System and 'pilot' for your car or truck. It features an amazing auto-drive system as well as manual corrections. This project features a custom trained model and vision. This system is made to be redundant and safe.

Note: Issues can always happen, use at your own risk. It is recommended to test it first using your car or van model in our simulator, soon.
